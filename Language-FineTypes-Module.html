resolveImports :: Map ModuleName Module -> Module -> Maybe Declarations Source #

For a given Module, resolve import by adding the declarations for the imported names.

Fails if an imported name cannot be found.

Collect all Typ names that have not been defined or imported in the Module.

Resolve all variables in a Typ that can be resolved using the given declarations.

This function will loop in the case of recursive definitions.
